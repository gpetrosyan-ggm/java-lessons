2. Սարքել class User, 3-5 փոփոխականերով
3. Սարքել class Book, 3-5 փոփոխականերով
4. Բոլոր class֊ների օրինակները սարքել (new)

-----------------------------------------------------

1. Սարեքել կալկուլյատոր
    1. add մեթոդ (+)
    2. subtract մեթոդ (-)
    3. multiply մեթոդ (*)
    4. divide մեթոդ (/)

2. Սարքել Computer կլաս (ամենաքիչը 5 փոփոխական), պարտադիր 2-3 փոփոխականերով
    1. Տպել պարամետրերը
    2. Համեմատել հիշողությունը/արագությունը դրսից եկացող պարամետրի հետ

3. Մի հատ էլ ցանկացաց կլասս իր փոփոխականերով, ըստ քո ցանկությաբմ

-----------------------------------------------------

1. Կլասս ֊ պատմել ու օրինակի վրա ցույց տալ
2. Կլասսի օբեկտ ֊ պատմել ու օրինակի վրա ցույց տալ
3. Կոնստրոկտոր ֊ պատմել ու օրինակի վրա ցույց տալ
4. Մեթոդ ֊ պատմել ու օրինակի վրա ցույց տալ
5. Փոփոխական ֊ պատմել ու օրինակի վրա ցույց տալ
6. Պարամետեր ֊ պատմել ու օրինակի վրա ցույց տալ
7. Արգումենտ ֊ պատմել ու օրինակի վրա ցույց տալ
