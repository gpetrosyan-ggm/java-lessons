15.27 Փորձի num1 == 0 ստուգումը while֊ով անես (բայց արդեն ուրիշ ստուգում կլինի)
      պարտադիր չի միանիշ լինի որ պազիցիան գտնես, թվի թվանշաները դեմից գալուց հենց հանդիպեց, դա էլ հենց իրա պազիցիանա,
      quantity կարաս վերջում տպես ու տպելուց տեղի թիվը հակառակ ուղղությամբ տեղի թիվն ես տպել։

15.28 Փորձի num == 0 ստուգումը while֊ով անես (բայց արդեն ուրիշ ստուգում կլինի), տվյալ դեպքում 0֊նելա թիվ
