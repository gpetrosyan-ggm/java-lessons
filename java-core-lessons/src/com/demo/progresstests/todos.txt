18.5 Դզել (երբ n-ը կենտ է)։

18.8 Դզել:

18.10 Աղյուսակում b[4] մոռացել ես, բայց մնացածը ճիշտա։

18.14 Ես պայմանը չես կատարել
    • в первой строке - значения из четных ячеек массива;
    • во второй строке - значения из нечетных ячеек массива.

18.16 c փոփոխականը մտցրաա for֊ի մեջ, կամ կարաս վաբշե հանես (առանց ավելորդ փոփոխականի ել կստացվի,
                                                            բայց եթե քեզ հարմարա c֊ով, մտցրա for֊ի մեջ)

18.15 Ես պայմանը չես կատարել
    • в первой строке - четные значения, содержащиеся в массиве;
    • во второй строке - нечетные значения, содержащиеся в массиве.
